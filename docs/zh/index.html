个人理财系统的基石：三大支柱

用一套简单、可用一生的系统，构建稳健的个人财务 —— 浓缩于一份简短的免费白皮书。提供六种语言。

收入 − 支出 = 自由基金

三大支柱

01

现金流

掌握收支，揭示你真正的自由基金这一根基。

02

投资

让金钱为你工作，借助复利随时间增长。

只投资于你理解的事物。

03

储蓄

持有保值资产，对抗通胀、降低人生风险。

安装后你的 AI 会有什么变化

  • 在给出建议前，先以你的真实数字为锚 —— 收入 − 支出 = 自由基金。
  • 拒绝炒作：不推荐你看不懂的资产。
  • 始终在计划中同时保留进攻（投资）与防守（储蓄）。
  • 注重纪律与 5 至 10 年的长期视野，而非聪明的择时操作。

你的提问将获得更精准、更一致、更具针对性的财务指导。

如何使用

🤖 AI 方式 — 安装技能

本白皮书同时提供 AI skill —— 一套推理框架，任何有能力的 AI 都可以应用。一条命令即可安装，或粘贴一个文件。

⚡ 自动安装（一条命令）

Claude Code — plugin

/plugin marketplace add nontravis/personal-finance-whitepaper，然后 /plugin install three-pillar-finance@nontravis。更新：/plugin marketplace update nontravis，然后 /reload-plugins。

Claude Code — degit

npx degit nontravis/personal-finance-whitepaper/skill ~/.claude/skills/three-pillar-finance。更新：加上 --force 重新运行。

CLI agents (Gemini CLI, Copilot CLI)

将 skill/ 文件夹复制到 agent 的适配器目录或 AGENTS.md；加上 --force 更新。

✋ 手动安装（copy-paste）

claude.ai / ChatGPT / Gemini / API

将 three-pillar-lens.md 粘贴到 Project、Custom Instructions、Gem 或 system prompt 中。如需更新，重新粘贴最新文件。
