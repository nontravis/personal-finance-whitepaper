パーソナルファイナンス・システムの基盤：3本の柱

シンプルで一生使える一つの仕組みで、揺るがない家計を築く —— 短い無料の白書にまとめました。6言語で提供。

収入 − 支出 = 自由資金

3本の柱

01

キャッシュフロー

収支を把握し、本当の自由資金という土台を明らかにする。

02

投資

お金に働いてもらい、複利で時間をかけて育てる。

理解できるものにだけ投資する。

03

貯蓄

価値を保つ資産を持ち、インフレに対抗しリスクを減らす。

このスキルをインストールするとAIがどう変わるか

  • アドバイスの前に、あなたの実際の数字 —— 収入 − 支出 = 自由資金 —— を必ず起点にする。
  • ハイプを拒絶する：理解できない資産は勧めない。
  • 常に攻め（投資）と守り（貯蓄）の両方をプランに組み込む。
  • 巧みなタイミング狙いより、規律と5〜10年の長期視点を優先する。

あなたの質問に対し、より鋭く、より一貫した、より具体的な財務アドバイスが返ってきます。

使い方

🤖 AIでの使い方 — スキルをインストール

この白書は AI skill としても提供されています —— あらゆる有能な AI が適用できる推論レンズです。コマンド一つでインストールするか、ファイルを一枚貼るだけ。

⚡ 自動インストール（コマンド一つ）

Claude Code — plugin

/plugin marketplace add nontravis/personal-finance-whitepaper、次に /plugin install three-pillar-finance@nontravis。更新: /plugin marketplace update nontravis、次に /reload-plugins。

Claude Code — degit

npx degit nontravis/personal-finance-whitepaper/skills/three-pillar-finance ~/.claude/skills/three-pillar-finance。更新: --force を付けて再実行。

CLI agents (Gemini CLI, Copilot CLI)

skill/ フォルダを agent のアダプタディレクトリまたは AGENTS.md に配置する。更新は --force を付けて再実行。

✋ 手動インストール（copy-paste）

claude.ai / ChatGPT / Gemini / API

three-pillar-lens.md を Project のカスタム指示、Custom Instructions、Gem、または system prompt に貼り付ける。更新するには最新ファイルを貼り直す。
