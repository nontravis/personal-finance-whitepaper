Fondasi Sistem Keuangan Pribadi: Tiga Pilar

Bangun keuangan pribadi yang kokoh dengan satu sistem sederhana seumur hidup — dalam whitepaper singkat dan gratis. Tersedia dalam enam bahasa.

Pemasukan − Pengeluaran = Dana Kebebasan

Tiga pilar

01

Arus Kas

Ketahui uang masuk dan keluar — fondasi yang mengungkap Dana Kebebasan Anda.

02

Investasi

Biarkan uang bekerja untuk Anda dan tumbuh lewat bunga majemuk.

Investasikan hanya pada apa yang Anda pahami.

03

Menabung

Simpan aset yang menjaga nilai untuk melawan inflasi dan mengurangi risiko.

Apa yang berubah pada AI Anda setelah menginstal ini

  • Mendasarkan setiap jawaban pada angka nyata Anda — Pemasukan − Pengeluaran = Dana Kebebasan — sebelum memberi saran.
  • Menolak hype: tidak akan merekomendasikan aset yang tidak Anda pahami.
  • Selalu menjaga keseimbangan antara ofensif (Investasi) dan defensif (Menabung) dalam rencana.
  • Mengutamakan disiplin dan cakrawala 5–10 tahun daripada menebak waktu pasar.

Pertanyaan Anda akan mendapat panduan keuangan yang lebih tajam, konsisten, dan tidak generik.

Cara menggunakannya

🤖 Cara AI — instal skill-nya

Whitepaper ini juga hadir sebagai AI skill — lensa penalaran yang dapat diterapkan oleh AI mana pun. Instal dengan satu perintah, atau tempelkan satu file.

⚡ Instal otomatis (satu perintah)

Claude Code — plugin

/plugin marketplace add nontravis/personal-finance-whitepaper lalu /plugin install three-pillar-finance@nontravis. Perbarui: /plugin marketplace update nontravis lalu /reload-plugins.

Claude Code — degit

npx degit nontravis/personal-finance-whitepaper/skill ~/.claude/skills/three-pillar-finance. Perbarui: jalankan ulang dengan --force.

CLI agents (Gemini CLI, Copilot CLI)

Salin folder skill/ ke direktori adapter agen atau ke AGENTS.md; perbarui dengan --force.

✋ Manual (copy-paste)

claude.ai / ChatGPT / Gemini / API

Tempel three-pillar-lens.md ke dalam Project, Custom Instructions, Gem, atau system prompt. Untuk memperbarui, tempel ulang file terbaru.
